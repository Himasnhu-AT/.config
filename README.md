# .config

my config for nvim


## Install

#### Run `setup.sh`

setup.sh will install important packages like:
- git
- curl
- plug.vim

#### Then open `init.vim` using neovim and run command:
:PlugInstall
to install all plugins and restart nvim

## Commands:
- :NERDTree : get tree structure of you repo to browse through
